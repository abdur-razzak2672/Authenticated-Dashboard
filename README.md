# Installation & Run

1. Copy environment example:
   ```bash
   cp .env.local.example .env.local
   ```

2. Open `.env.local` and assign values (API base URL, API key, etc.).

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start dev server:
   ```bash
   npm run dev
   ```
